# leetcode  
[leetcode](https://leetcode-cn.com)题目解法，除了一些用 C 实现起来太复杂的题目，我用其它语言实现，基本采用 C 进行编码，由于 C 缺乏一些高级的数据结构，因此有些题目的实现在时间复杂度上可能不会是最优的解法，一般不会超过O(n^2)。  

题目的解法均放在`solution`目录下，按照题目的编号进行命名，文件开头注明了题目名称和链接。
